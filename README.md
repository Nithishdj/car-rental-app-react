## üìå Project Overview
AutoRent is a user-friendly platform that connects customers with a fleet of rental vehicles. It features a responsive interface for browsing cars, managing bookings, and a dedicated admin panel for fleet management. The application focuses on providing a seamless experience from registration to vehicle return.

### üåü Key Features
* **User Authentication:** Secure Login and Registration system.
* **Vehicle Browsing:** Filter cars by category (Sedan, SUV, Luxury) and price.
* **Booking System:** * Select Pick-up and Drop-off dates.
    * Automated price calculation based on rental duration.
    * **Loyalty Program:** Earn ‚≠ê points on every booking and redeem them for discounts.
* **User Dashboard:** View booking history, manage profile details, and track loyalty points.
* **Admin Dashboard:** * Add, Edit, and Delete cars from the fleet.
    * Manage user bookings (Approve/Reject).
    * View total revenue and system statistics.
* **Responsive Design:** Optimized for both desktop and mobile devices.

4.  **CRUD Operations:** Interacting with a REST API (JSON Server) to Create, Read, Update, and Delete data for cars and bookings.

6.  **UI/UX Design:** Creating modern, card-based layouts and responsive tables using CSS Flexbox and Grid.
7.  **Version Control:** Using **Git & GitHub** for source code management and collaboration.

## ‚öôÔ∏è How to Run This Project

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/your-username/your-repo-name.git](https://github.com/your-username/your-repo-name.git)
    ```

2.  **Navigate to the project directory:**
    ```bash
    cd your-repo-name
    ```

3.  **Install dependencies:**
    ```bash
    npm install
    ```

4.  **Start the JSON Server (Backend):**
    ```bash
    npx json-server --watch data/db.json --port 8000
    ```

5.  **Start the React App (Frontend):**
    Open a new terminal and run:
    ```bash
    npm start
    ```

6.  **Access the App:**
    Open [http://localhost:3000](http://localhost:3000) in your browser.
